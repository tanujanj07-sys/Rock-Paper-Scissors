# Rock-Paper-Scissors
Rock Paper Scissors Multiplayer is a fully responsive web game built with vanilla HTML, CSS, and JavaScript. It features both Player vs Player and Player vs Computer modes with real-time turn-based gameplay, dual timers, and smooth animations.
🎮 Key Features

    Dual Game Modes: Player vs Player (local multiplayer) and Player vs Computer

    Real-time Turn System: 5-second turn timer + 2-minute game duration

    Visual Feedback: Emoji choice animations with 360° spin effects

    Score Tracking: Live scoreboard with final winner declaration

    Responsive Design: Mobile-first layout with touch-friendly controls

    Glassmorphism UI: Modern gradients, backdrop blur, and hover effects

    Timeout Handling: Automatic turn switching for inactive players

🚀 Game Flow

text
1. Mode Selection → Player vs Player / Computer
2. Enter Names → Start Game
3. Player 1 chooses → Player 2's turn (5s timer)
4. Result shown → Scores updated → Next round
5. Game ends at 2:00 → Final winner declared
